🔑 API 키 발급 가이드

Google AI Studio에서 무료 API 키를 발급받아 사용하세요. (약 2분 소요)

1
Google AI Studio 접속

👉 aistudio.google.com/apikey 에 접속합니다.
Google 계정으로 로그인하세요.

2
API 키 생성

Create API Key 버튼을 클릭합니다.
프로젝트를 선택하거나 새로 생성하세요.

3
API 키 복사

생성된 키 (예: AIzaSy...)를 복사합니다.

4
여기에 붙여넣기

상단의 API Key 입력란에 키를 붙여넣고 저장 버튼을 누르세요.
키는 브라우저에 안전하게 저장되며, 다음에 접속해도 유지됩니다.

💡 참고: Google AI Studio의 Gemini API는 무료 등급이 제공됩니다. 일일 사용량 제한이 있지만 일반적인 사용에는 충분합니다. API 키는 본인만 사용하며, 다른 사람과 공유하지 마세요.
